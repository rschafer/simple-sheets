# Claude Code Resource Reference

Complete documentation of all available commands, skills, and agents for this project.

## Available Commands

### Planning & Project Management
- `/plan` - Generate PRD and execution plan
- `/spec` - Comprehensive specification analysis
- `/build` - Build from transcript (frontend & backend analysis)
- `/execute` - Run your execution plan with parallel subagents
- `/todo` - Organize and manage tasks

### Development Tools
- `/resolve_parallel` - Run operations in parallel
- `/deploy-docs` - Deploy documentation
- `/research` - Conduct research tasks

### Content & Communication
- `/book` - Book-related operations
- `/yt-transcript` - Extract YouTube transcripts
- `/cc-research` - Analyze r/ClaudeCode discussions
- `/email` - Draft professional emails

## Quick Reference

### Invoke a Command
```bash
/plan
```

### Invoke a Skill
```bash
/skill-name
```

### Launch an Agent
Use the Task tool with `subagent_type` parameter:
- `general-purpose` - Research, searching, multi-step tasks
- `Explore` - Fast codebase exploration
- `Plan` - Architecture and implementation planning

## Project Files

- `docs/transcript.md` - Your project's single source of truth
- `docs/prd.md` - Generated by `/plan`
- `docs/todo.md` - Generated by `/plan` (execution plan)
- `docs/ui.md` - Generated by `/build` (frontend analysis)
- `docs/backend.md` - Generated by `/build` (backend analysis)

## Tips for Success

1. Keep `docs/transcript.md` updated with your project vision
2. Reference this file frequently for available resources
3. Use `/plan` to start your project workflow
4. Use `/execute` to build with agent assistance
5. Track progress with `/todo`
